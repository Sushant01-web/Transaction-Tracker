🚀 Features
💰 Admin can credit/debit client wallets
👤 Each client has an individual wallet
🧾 Maintains transaction ledger (credit/debit history)
🛒 Clients can create orders
🔄 Automatically calls external fulfillment API
📊 Stores fulfillment ID with order
🔍 Fetch order details & wallet balance

🏗️ Tech Stack
Backend: Node.js, Express.js
Database: MongoDB (Mongoose)
API Calls: Axios

Insatallation
npm install
